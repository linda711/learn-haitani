# learn.haitani.org

Video series landing page — deployed via Vercel at `learn.haitani.org`.

## Before deploying

Add these two image files to this folder:

- `linda-haitani.jpg` — headshot photo (used in About section)
- `video3-thumbnail.jpg` — Video 3 thumbnail image

## Deployment

Connected to Vercel. Push to `main` to deploy.

## To add a new video

1. Edit `index.html`
2. Find the Video 3 block and duplicate it
3. Replace the `media-id` with the new Wistia ID
4. Add the new Wistia embed script in the `<head>`
5. Push to `main`
